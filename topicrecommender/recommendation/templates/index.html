Latent Dirichlet Allocation. Topic Modeling Algorithm.

Latent Dirichlet Allocation (LDA) is a popular topic modeling algorithm in natural language processing. The aim of Latent Dirichlet Allocation is to find the different topics that a document belongs to, based on the words in the document. The Topic Recommender uses LDA to match your content with various topics.

Term Frequency-Inverse Document Frequency. Information Retrieval Algorithm.

Term Frequency-Inverse Document Frequency, more commonly known as TF-IDF, is a statistical algorithm that weighs how important a word is to a whole document or a set of documents. It considers how many times a word appears in a document, and the inverse document frequency of the word. The Topic Recommender uses TF-IDF to calculate the keywords sent to Google Trends.

PyTrends. Google Trends API.

PyTrends is an unnofficial API for Google Trends. It allows the user to access data from Google Trends when given a payload with one or more keywords. The Topic Recommender uses PyTrends to find "trending" content.
